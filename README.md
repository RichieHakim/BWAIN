# BWAIN: Brain-machine-interfacing Without An Interesting Name
Brain-machine-interfacing suite for 2-photon calcium imaging

Developed to be used with:
- Suite2p
- ScanImage + NIDAQ
- BMI_IDAP (see my repo)
- Teensy controller for feedback
